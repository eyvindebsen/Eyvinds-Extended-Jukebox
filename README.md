# Eyvind's Extended Jukebox Vol 00. V1.0.
85 high-quality SIDs using one disk side (D64).
~4+ hours of music.

How to use:
Grab joystick in port 2 to control the jukebox.
Joy up/down to choose a SID in the list. Press fire to play.
When SID is playing, press fire to get back to the menu.

In the menu you can use:

Joystick Left chooses a play mode:
  Continue : plays the SIDs one after another in the list.
  Random   : plays the selected SID. Plays random SIDs after that.
  Loop	   : plays the selected SID over and over, ignoring time length.  
	     
While the music plays, push the joystick down and it will:
If play mode is
  Continue  : skip to the next SID.
  Random    : pick a random SID.

Known quirks:
Several songlengths are not 100% correct. Due to some songs loops while others fade out. So when playing looped songs it will sound like somebody just pulled the plug.
You can avoid this by using the "loop" play mode.

FAQ: Where is the fastloader?
I wish i had the expertise to include a fastloader that would support all the new stuff out there (and not mess stuff up), but i don't. Maybe in Vol 01.
And since this should be a 80s jukebox-look-a-like thing, there is always a little wait when changing records.

Enjoy the music! :)
